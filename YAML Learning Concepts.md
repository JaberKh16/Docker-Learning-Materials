## A simple key-value pair
- syntax: key:pair
	name: Google Search

## A sequence (list) of items
fruits:
  - Apple
  - Banana
  - Orange

## A mapping (dictionary) with nested data
person:
  name: Alice
  age: 30
  is_student: false

## A multi-line string (literal block)
description: |
  This is a very
  long description that
  preserves line breaks.

## A multi-line string (folded block)
summary: >
  This is a folded block
  that will have its newlines
  replaced by spaces.
